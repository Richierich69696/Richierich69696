The Projects:
Most of my repositories sit at the intersection of cybersecurity and machine learning — not because AI is trending but because the problems I kept running into at work had ML-shaped solutions.

OSINT-Photo-Hunt — what a photograph quietly tells you without the owner knowing. Metadata, GPS, device fingerprints. Eye-opening exercise in how much information people give away without realizing it.

Cyber-Alert-Triage — the skill that separates good analysts from great ones is not detecting threats, it is knowing which alerts to ignore. This project practices exactly that.

CyberAnalyst-AI-Shadow — trained a model on analyst triage decisions so it can handle the repetitive classification work. The idea came from watching how consistent my own patterns were across hundreds of alerts.

AI-Driven-Vulnerability-Scout — Nmap gives you a list of open ports. This adds a layer that tells you which ones are actually worth losing sleep over.

AI-Boosted-Intrusion-Sniffer — anomaly detection on packet captures. Catches the slow, quiet reconnaissance that signature-based tools were never designed to see.

AI-Powered-Phishing-Detector — Naive Bayes classifier trained on phishing language patterns. Turns out attackers have very predictable vocabularies.

AI-Powered-Password-Cracker-Detector — brute force attacks have a distinctive rhythm. This detects that rhythm before it succeeds.

AI-Network-Spy-Spotter — live packet monitoring with behavioral thresholds. For catching what does not belong on a network before it does damage.

Wireshark-Credential-Analysis — a practical demonstration of why plaintext protocols are indefensible. Sometimes showing is more convincing than telling.

Network-Health-Check — because most organizations have no idea what is actually running on their network until something goes wrong.
